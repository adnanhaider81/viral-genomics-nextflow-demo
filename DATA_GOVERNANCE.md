# Data Governance

This repository is intended for public, reproducible workflow demonstration and method documentation.

- Do not commit patient identifiers, clinical records, private surveillance metadata, raw restricted FASTQ/BAM/VCF files, or unpublished operational datasets.
- Example inputs, when present, are synthetic, public, or placeholders for testing workflow structure.
- Real analyses should use local or private paths outside the repository, with neutral sample identifiers in configuration files, reports, screenshots, and figures.
- Generated outputs that may reveal restricted sequences, sample names, locations, dates, or laboratory metadata should remain outside git unless approved for public release.
- Before sharing results, confirm permission from the data owner, laboratory, institution, and any applicable public-health authority.
